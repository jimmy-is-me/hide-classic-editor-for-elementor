# Hide Classic Editor for Elementor Pages

Elementor 頁面強制使用 Elementor 編輯器，防止客戶誤用 WordPress 一般編輯器造成跑版。

## 功能

| # | 功能 | 說明 |
|---|------|------|
| 1 | 頁面列表按鈕 | 移除重複項，只留「使用 Elementor 編輯」 |
| 2 | 標題點擊連結 | 後台頁面標題直接進 Elementor |
| 3 | 上方管理列 | 「編輯頁面」按鈕改為導向 Elementor |
| 4 | 強制跳轉 | 誤入一般編輯器自動跳回 Elementor |
| 5 | CSS 隱藏 | 隱藏 Elementor 原生的重複按鈕 |

## 安裝

1. 下載此 repo
2. 上傳到 `/wp-content/plugins/`
3. 在 WordPress 後台啟用外掛
4. 確認已安裝 Elementor

## 需求

- WordPress 5.0+
- PHP 7.4+
- Elementor（任何版本）
